# Sorteador de Números

Um simples sorteador de números desenvolvido com **HTML, CSS e JavaScript**.  Este projeto permite ao usuário definir um intervalo (mínimo e máximo) e a quantidade de números a serem sorteados, gerando uma lista de números aleatórios **sem repetição** e exibindo-os em ordem crescente.



## 🚀 Funcionalidades

- Sorteio de números aleatórios dentro de um intervalo definido pelo usuário  
- Definição da quantidade de números a serem sorteados  
- Números sorteados sem repetição  
- Resultado exibido em ordem crescente  
- Interface simples e responsiva   

## 🛠️ Tecnologias Utilizadas

- **HTML5** – Estrutura da página  
- **CSS3** – Estilização e layout responsivo  
- **JavaScript** – Lógica do sorteio e interações   

## 📦 Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/JoseFelisbino/Sorteador-de-numeros.git
   ```

2. Abra o arquivo `index.html` em seu navegador.

3. Informe:
   - O número mínimo
   - O número máximo
   - A quantidade de números a sortear

4. Clique em **"Sortear"** e veja o resultado!

## 🎯 Exemplo

> Sortear 6 números entre 1 e 60  
> Resultado: `3, 12, 27, 34, 45, 58`

Ideal para uso em bolões de loteria, decisões aleatórias ou jogos!
